# Exercises

1. [Download the Simple Web App program](https://github.com/learning-go-book-2e/simplewebapp). Write unit tests for the program and get as close to 100% code coverage as
   you can. If you find any bugs, fix them.

2. Use the race detector to find a concurrency problem in the program and fix
   it.

3. Write a fuzz test against the parser function and fix any problems that you
   find.
